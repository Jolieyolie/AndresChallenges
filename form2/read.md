Forms and Values_Level_02

# Task

- Learning objective: After completing this exercise, you will be able to process user input and change the content of a web page (DOM manipulation).
- In this task, you will create a form to calculate your age
- To do this, create a form in which you can enter your year of birth, as well as a button that executes the function
- Define a function that outputs your age in years. Your age should then be output in your HTML

# Note

- Add onsubmit="return false" to your form if your page reloads after submission.
