  1. Function 1: Declare the function intro2() with the parameter (paramName). The following then happens in the function body {}: Declare the variable name with the value “User”. The output on the console should be: Hi User. My name is paramName
  2. Function 2: Declare the function intro3() with the three parameters (name, city, age). Using the appropriate arguments, output the following sentence in the console: “Hello, my name is Max. I am 27 years old. I come from Hamburg.”
  3. Function 3: Declare the function math() with two parameters.The following happens in the function body {}: Output the multiplication of the two parameters in the console.Output the division of the two parameters in the console.
