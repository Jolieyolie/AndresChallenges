Forms and Values_Level_03

# Task

- Learning objective: After completing this exercise, you will be able to process user input with JavaScript
- Create a form to compare the ages of two people
- Use two input fields, a button that calls the function, and a function that writes the difference in years to your HTML

# Note

- Add onsubmit="return false" to your form if your page reloads after submission.
- Try to think in a user-friendly way.
