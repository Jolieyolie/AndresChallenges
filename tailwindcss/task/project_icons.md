# SVG Icons for Tailwind CSS Lesson

## üìå How to Use These Icons

Copy the SVG code and paste it directly into your HTML where you need an icon.

**Example:**

```html
<div class="w-6 h-6 text-blue-600">
  <!-- Paste SVG code here -->
  <svg class="w-6 h-6" fill="none" stroke="currentColor" viewBox="0 0 24 24">
    <path
      stroke-linecap="round"
      stroke-linejoin="round"
      stroke-width="2"
      d="M13 10V3L4 14h7v7l9-11h-7z"
    ></path>
  </svg>
</div>
```

**Key Points:**

- `class="w-6 h-6"` controls the size
- `fill="none"` and `stroke="currentColor"` make icons inherit text color
- Use `text-blue-600` (or any color) on parent div to change icon color
