# Task

- Learning objective: Application and syntax of if/else statements in JavaScript to integrate conditional logic into the code.
- Create an HTML form for entering the weather quality as a number between 0 and 10.
- Declare the function weather(), use conditional statements (if...else) in the function body, and output the results in HTML:
- For a quality between 8 and 10, the result should be ‘great’. For a quality between 6 and 7, the result should be ‘good’. For a quality between 3 and 5, the result should be ‘okay’. For a quality between 0 and 2, the result should be ‘poor’.
