# Task

- Learning objective: Application and syntax of if/else statements in JavaScript to integrate conditional logic into the code.
- We need a query to check whether the user is of legal age. Declare the function adult().
- Create an HTML file with a form for entering the age and a button for verification.
- In the function body:
- If the entered age is greater than or equal to 18, return true.
- If the entered age is less than 18, return false.
- Display the result of the function in the HTML as “Adult” or “Minor.”
